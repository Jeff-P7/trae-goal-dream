Design a modern mobile application UI for a dream and goal tracking platform.

The app helps users turn dreams into actionable systems by creating goals, milestones, and tasks. The design should feel similar in quality and layout to a premium fitness or habit tracking app, with a clean, soft, and motivating aesthetic.

STYLE & VISUAL DIRECTION:
- Soft UI (neumorphism-inspired but subtle)
- Rounded cards and containers (16–28px radius)
- Light neutral background (off-white / light gray)
- Pastel accent colors (green, blue, peach)
- Smooth shadows (soft elevation, not harsh)
- Minimal but highly structured layout
- Modern typography (clean sans-serif, medium weight hierarchy)
- Use whitespace intentionally for clarity and calmness

---

## 📱 SCREENS TO DESIGN

### 1. ONBOARDING / LOGIN SCREEN
- Full-screen background image (motivational lifestyle image)
- Headline: “Build Your Dream Life”
- Subtext: “Turn goals into daily actions”
- Two buttons:
  - “Continue with Google”
  - “Get Started”
- Bottom text: “Already have an account? Sign in”

---

### 2. DASHBOARD (HOME)
- Top section:
  - Profile avatar + greeting (“Hello, [Name]”)
  - Notification icon

- Date selector (horizontal calendar like habit tracker)
- Main cards:
  - “Active Goals” (progress % circle)
  - “Today’s Focus” (top tasks for the day)
  - “Consistency / Streak” card

- Weekly progress chart (bar graph style)
- Summary stats:
  - Tasks completed
  - Milestones completed
  - Productivity score

---

### 3. GOAL DETAIL SCREEN
- Header:
  - Back button
  - Goal title
  - Options (3 dots)

- Goal progress ring (large circular progress indicator)
- Info cards:
  - Target date
  - Priority
  - Category

- Sections:
  1. Milestones (card list)
     - Each milestone shows:
       - Title
       - Progress %
       - Small progress bar
  2. Direct Tasks (tasks not tied to milestones)

- Floating “+ Add” button

---

### 4. MILESTONE DETAIL SCREEN
- Title + progress bar
- List of tasks:
  - Checkbox
  - Title
  - Due date
  - Priority indicator
- Completion percentage at top

---

### 5. TASK CREATION MODAL
- Modal or bottom sheet
- Fields:
  - Task name
  - Description
  - Due date picker
  - Priority selector (Low / Medium / High)
  - Assign to milestone (optional dropdown)
- CTA: “Create Task”

---

### 6. CREATE GOAL / DREAM FLOW (WIZARD)
Multi-step flow:
1. Goal title + description
2. “Why does this matter?” (motivation input)
3. Target date + category
4. Suggested milestones (AI-style suggestions UI)
5. Review & create

Use step indicators at top.

---

### 7. ANALYTICS / PROGRESS SCREEN
- Large progress charts:
  - Weekly activity
  - Goal completion trends
- Stats:
  - Completion rate
  - Consistency score
- Clean graph cards (rounded, soft colors)

---
